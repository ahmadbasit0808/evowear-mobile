# evowear-mobile

**Libraries and their usage**

- expo-image-picker (_used to add image picker_)
- react-native-elements (_used to add badges(icon popups)_)
- react-native-paper (_used to add ready-made components_)
- react-native-swiper (_used to add image carousel_)
- react-native-navigation/native-stack (_used to add stack navigation_)
- react-native-navigation/bottom-tabs (_used to add tab navigation_)
- react-native-navigation/drawer (_used to add drawer navigation_)
